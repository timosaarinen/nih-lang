int mandelbrot( vec2 c )
Why?
Calculates the number of iterations required to determine whether a given point \( c = c_x + c_yi \) in the complex plane is part of the Mandelbrot set.
Input?
c vec2 - The complex number represented as a 2D vector, where c.x is the real part and c.y is the imaginary part.
Output?
int The number of iterations before the sequence escapes the radius of 2 around the origin, or 0 if the sequence does not escape within the maximum number of iterations.
Details?
In the Mandelbrot set, for each point \( c = c_x + c_yi \), the function iterates \( Z_{n+1} = Z_n^2 + c \) starting from \( Z_0 = 0 \) and determines whether the sequence remains bounded. If the magnitude of \( Z_n \) exceeds 2 at any iteration, the sequence will escape to infinity, and the point \( c \) is not in the Mandelbrot set. The number of iterations before this escape is what we typically visualize in Mandelbrot set images.
